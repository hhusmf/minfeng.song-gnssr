Python C Extension for Dislocation Analysis
March 2025
Python, C
Research Software

This Python C extension implements classical Okada and TDEs (Triangular Dislocation Elements) algorithms for efficient computation of surface displacements caused by subsurface dislocations. The implementation is optimized for large-scale analyses with significant performance improvements over pure Python implementations.

Key features include:

  • Fast C implementation of Okada's solutions
  • Support for triangular dislocation elements
  • Parallelized computation for large datasets
  • Python interface with NumPy integration

Earthquake Data Analysis Toolkit
November 2024

Data Analysis

A comprehensive Python package for processing and analyzing earthquake-related datasets. This toolkit provides functions for data cleaning, statistical analysis, visualization, and modeling of seismic data.

Main components:

InSAR Processing Pipeline
June 2023
Python, MATLAB
Remote Sensing

An end-to-end processing pipeline for Interferometric Synthetic Aperture Radar (InSAR) data analysis. The pipeline automates the process from raw data to displacement time series and includes tools for noise reduction and deformation modeling.

Features:

  • Automatic atmospheric correction
  • Time series analysis with SBAS algorithm
  • Integration with geophysical modeling tools
  • GIS-compatible output formats
